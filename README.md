# Weather App

A simple frontend weather application that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Search weather by city name
- Displays current temperature and conditions
- Uses the OpenWeatherMap REST API
- Built with vanilla HTML, CSS, and JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How It Works
The app sends a request to the OpenWeatherMap API using a city name entered by the user.  
The API responds with weather data, which is then displayed on the page.
